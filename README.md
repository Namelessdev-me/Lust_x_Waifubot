## 💀 *Features*
• Waifu character collection system  
• Multiple rarities & themed editions  
• Harem system to view and flex your collection  
• Trade and gift waifus with other users  
• Global and chat-wise leaderboards  
• Economy system with coins and payments  
• Stylish anime-themed interface  


## **📤 How to Upload Waifus*

(Admin / Uploader only — reply to an image)

Command:
/upload <character-name> <anime-name> <rarity-number>

Example:
/upload zero-two darling-in-the-franxx 8

## *🎮 User Commands*

/Fetch     - Guess the character  
/fav        - Add waifu to favorites  
/harem      - View your waifu collection  
/trade      - Trade waifus  
/gift       - Gift waifu to another user  
/tops       - Global leaderboard  
/ctop       - Chat leaderboard  
/pay        - Pay coins  
/changetime - Change spawn time  



## *🛠️ Admin / Uploader Commands*

/upload     - Upload waifu  
/delete     - Delete waifu  
/update     - Update waifu details  
/addsudo    - Add bot developer  
/sh         - Use bash commands

## 🔧 **Deployment Guide**
1. **Clone the Repository:**  
 
```
   bash git clone https://github.com/Harsh04756/fetcher_x_waifubot.git
```

2. Install Dependencies:
```
pip install -r requirements.txt
```

3. Set Environment Variables:
Create a .env file and add the following details

```
BOT_TOKEN = your_telegram_bot_token
MONGO_URI = your_mongodb_connection_uri
OWNER_ID = your_telegram_user_id
LOG_CHAT_ID = your_log_channel_id
```

4. Run the Bot:
```
python3 m- Lust 
```
